# 🔤 Live Input Filter

A beginner-friendly JavaScript project that filters user input in real time. As the user types, only alphabetic characters (A-Z, a-z) and spaces are displayed, while numbers and special characters are ignored.

## ✨ Features

- Real-time Input Filtering
- Allows Only Letters and Spaces
- DOM Manipulation
- Event Handling
- Clean User Interface
- Bootstrap Input Group

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5

## 🚀 How to Run

1. Clone this repository.
2. Open `index.html` in your browser.
3. Type into the textarea.
4. Only letters and spaces will appear in the output.
